## 🚀 快速开始

### 1. 启动开发服务器
```bash
npm run dev
```
项目将在 http://localhost:3000 启动

### 2. 配置后端API
编辑 `.env` 文件：
```env
VITE_API_BASE_URL=http://localhost:3099
```

### 3. 生产构建
```bash
npm run build
```
